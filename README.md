# walk.io-base

Build and release vm base components for the [walk.io](https://github.com/maxdollinger/walk.io) project with a custom Linux kernel, rootfs and a firecracker binary.

## What's Included

- **Linux Kernel** - Built from the latest kernel tag (or custom tag)
- **Rootfs** - Minimal ext4 filesystem with init system
- **Firecracker Binary** - Latest Firecracker release
